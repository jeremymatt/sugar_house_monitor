Sugar House Monitor API

Available endpoints:

  • POST /sugar_house_monitor/api/ingest_tank.php – upload processed tank readings
  • POST /sugar_house_monitor/api/ingest_pump.php – upload processed pump events
  • POST /sugar_house_monitor/api/reset.php – truncate server DBs/status (debug only)

Each request must include the shared API key via the X-API-Key header and provide a JSON array.

Successful ingest will trigger scripts/process_status.py which regenerates web/data/status.json.
